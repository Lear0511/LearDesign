# フォーマット別の型

元コンテンツから「核となる主張」を1つ、「重要なインサイト」を3〜5個抽出してから展開する。

## 1. Xスレッド（12ツイート / 各280文字以内）
- 1本目はフック。スレッドを読む理由を1文で。
- 2〜11本目で1ツイート1アイデア。改行と空白で読みやすく。
- 12本目はまとめ＋CTA（フォロー・全文リンク等）。
- 絵文字は使うとしても最小限。文字数は280以内を厳守。

## 2. LinkedIn投稿（200〜300語）
- 冒頭2行で引き込む（プレビューで切れる位置を意識）。
- 短い段落と箇条書きで可読性を上げる。
- プロフェッショナルだが堅すぎない。一人称の学びを混ぜる。
- 末尾に問いかけや議論を促す1文。

## 3. ニュースレター導入文（100語）
- パーソナルで限定感のあるトーン。
- 全文を読みたくさせるティーザー。結論は出し切らない。
- 「続きはこちら」に自然につながる1文で締める。

## 4. SNS投稿3本（各独立）
- 3本それぞれ異なるインサイトを1つずつ主役にする。
- 単体で意味が通るようにする（スレッド前提にしない）。
- 短く、1投稿1メッセージ。

## 5. ショート動画台本（60秒 / 会話調）
- 最初の3秒でフック（視聴維持率が決まる）。
- カメラに向かって話す前提の口語。一文を短く。
- [画面テキスト] や [間] などの簡単なト書きを添える。
- 最後に明確なCTAで締める。
