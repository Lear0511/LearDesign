# content-repurposing — マルチフォーマット展開スキル

1つの元コンテンツを、各プラットフォームに自然に馴染む複数フォーマットへ展開する。

## いつ使うか
- 記事・投稿・文字起こしを別フォーマットに転用する依頼
- 「Xスレッド／LinkedIn／ニュースレター／ショート動画台本にして」系

## 入力（不足していれば必ず確認する）
- **元コンテンツ**（記事・投稿・文字起こし等）。空なら展開できないので必ず確認する。
- 展開先フォーマットの指定（未指定なら下記5種を既定にする）

## 作成するもの（既定の5フォーマット）
1. **Xスレッド**: 12ツイート、各280文字以内
2. **LinkedIn投稿**: 200〜300語、プロフェッショナルだが堅すぎない
3. **ニュースレター導入文**: 100語、全文へのティーザー
4. **SNS投稿3本**: それぞれ独立、それぞれ異なるインサイトをハイライト
5. **ショート動画台本**: 60秒、会話調、カメラに向かって話す想定

## ルール（厳守）
- 各フォーマットがそのプラットフォームに自然に馴染むこと。コピペ感を出さない
- 核となる主張と重要なインサイトは全フォーマットで維持する
- トーンはプラットフォーム別に調整する（下記）

## プラットフォーム別トーン
- **X**: パンチがあってストレート
- **LinkedIn**: プロフェッショナルで思慮深い
- **ニュースレター**: パーソナルで限定感

詳細な各フォーマットの型は `references/format-specs.md`、出力体裁は `assets/repurpose-template.md` を参照。
