# 形式 — format-templates

依頼された形式の構造に沿って書く。

## 記事
1. **フック**: 大胆な主張か意外な事実で始める。定義始まりはNG。
2. **問題提起**: 一般的な考え方がなぜ不完全か。
3. **本論**: 3〜5セクション。各見出しは具体的。各セクションに根拠（事例・データ）1つ以上。
4. **アクション**: 読者が「今週中に」できる具体策3つで締める。
- 各セクションで最重要の1文を **太字** に。

## メール
- **件名**: 具体的かつ行動を促す（「フォローアップ」等はNG）。
- **冒頭**: 最初の1文で本題。社交辞令なし。
- **本文**: 短い段落3つまで。各段落の役割は1つ。
- **締め**: 明確な次のアクションか依頼。
- 全体150語以内が目安。

## SNSスレッド（X等）
- 1本目はフック。読む理由を1文で。
- 2本目以降は1投稿1アイデア。各280文字以内。
- 最後はまとめ＋CTA。

## SNS単発投稿
- 1投稿1メッセージ。単体で意味が通る。
- 冒頭2行で引き込む。

> 各形式の詳しい雛形は、用途別スキル（content-writing / email-writing /
> content-repurposing 等）の assets/ も流用できる。
