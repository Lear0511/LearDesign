# 文脈 — audience

誰に向けて書くかを最初に確定する。未指定なら執筆前に確認する。

## 確認する項目
- **属性**: 職種・役割・立場（例: 中小企業のマーケ担当）
- **知識レベル**: 初心者 / 実務者 / 専門家
- **読む状況**: いつ・どこで・どんな目的で読むか
- **読後のゴール**: 読んだ後に何ができる／どう感じてほしいか

## 知識レベル別の調整
- **初心者**: 前提から説明。専門用語は定義＋アナロジー。一歩ずつ。
- **実務者**: 前提は省き、再現可能な手順と判断基準を重視。
- **専門家**: 既知の説明を省き、新規性・反証・限界に踏み込む。

## 書き分けの指針
- 読者が既に知っていることは書かない（冗長は離脱を生む）。
- 読者の語彙・関心・痛みに合わせて具体例を選ぶ。
- 1コンテンツ1ペルソナ。複数読者に同時に媚びない。
