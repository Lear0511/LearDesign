# 品質基準 — quality-checklist

公開前に全項目をチェックする。1つでもNGなら書き直す。
「編集なしでそのまま公開できる」が合格ライン。

## 制約
- [ ] 1段落3文以内を守っているか
- [ ] フィラー表現がゼロか
- [ ] ヘッジワードがゼロか

## 中身
- [ ] すべての主張が具体的か（数字・固有名詞・検証可能性）
- [ ] 事実を脚色で歪めていないか
- [ ] 各セクション／段落に存在理由があるか（冗長がないか）

## 読者適合
- [ ] 想定読者の知識レベルに合っているか
- [ ] 読者が既知のことを冗長に説明していないか
- [ ] 読後のゴールに到達できる構成か

## 形式
- [ ] 指定形式の構造を満たしているか
- [ ] フック／件名が具体的で行動を促すか
- [ ] 締めに明確な次のアクションがあるか

## 仕上げ
- [ ] 誤字脱字・表記ゆれがないか
- [ ] 声に出して読んで詰まらないか
