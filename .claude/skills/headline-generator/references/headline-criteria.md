# 見出しの型と評価基準

## 4つの型
- **好奇心型**: 情報ギャップを作る。答えを言わず、知りたくさせる。煽りすぎは逆効果。
- **ベネフィット型**: 読者が得る結果を冒頭で明示する。動詞＋成果で書く。
- **逆張り型**: 一般常識に反する主張。根拠が本文にある場合のみ使う（釣りで終わらせない）。
- **数字型**: 具体的な数量・期間・割合を入れる。奇数や具体値は信頼感が出やすい。

## 予測クリック率（1〜10）の評価軸
1. 具体性: 数字・固有名詞があるか
2. 好奇心ギャップ: 続きを知りたくなるか
3. ベネフィット明示: 読む理由が伝わるか
4. ターゲット適合: 想定読者に刺さるか
5. 信頼性: 釣り感がなく本文と整合するか

各軸を加味して1〜10を付け、理由を1〜2文で書く。

## ありがちなNG
- 抽象的すぎる（「◯◯の基本」「◯◯について」）
- 釣りだけで本文と乖離する
- 長すぎて要点がぼける（目安40文字前後まで）

## TOP3の選び方
- 型の多様性も考慮し、最も予測クリック率が高い3本を選ぶ。
- なぜその順位かを、上記評価軸に紐づけて説明する。
