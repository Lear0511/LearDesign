# アウトライン作成の方法

## 手順
1. ゴールを1文で確定する（読了後に読者ができるようになること）。
2. ゴールから逆算してセクションを3〜6個に分解する。
3. 各セクションに見出し・要約・根拠・つなぎを割り当てる。
4. 目標文字数をセクションに配分する（導入と結論は短め）。
5. 見出し・メタ・リンク候補を最後に整える。

## 見出しの基準
- 具体的な名詞・数字・結果を含める。
- NG例:「◯◯の基本」「◯◯とは」/ OK例:「◯◯で離脱率を42%下げた3手順」
- セクション見出しは記事タイトルと重複させない。

## 代替見出し案のランク付け
- 3案出し、予測クリック率の高い順に並べる。
- ランク根拠を1行添える（数字入り／好奇心ギャップ／ベネフィット明示など）。

## メタディスクリプション
- 160文字以内。主要キーワードを前半に。
- 検索結果で読者が得る価値を1文で示す。

## リンク候補5件
- 内部リンク: 関連する自サイト記事（テーマで補完するもの）。
- 外部リンク: 一次情報・統計・権威あるソース。
- 各候補に「なぜ貼るか（根拠補強／用語定義など）」を1行添える。
